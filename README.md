Mystery Message is an anonymous messaging platform built with Next.js. A user signs up using email, username, and password, then verifies their account via an OTP sent to email. After verification, the user logs in and is redirected to a dashboard where they receive a unique public link.
Anyone can use this link to send anonymous messages without creating an account. The logged-in user can control message reception using a toggle switch. If enabled, messages are accepted and stored securely; if disabled, submissions are blocked.
The public page also includes an AI-powered message suggestion feature that helps users generate messages. All received messages appear in the dashboard, sorted by latest, where the user can manage or delete them.
